# Installation


``git clone https://github.com/cta-sst-1m/histogram``
``cd histogram/``
``pip install -r requirements.txt``
``pip install .``

## Requirements

See the file : ``requirements.txt``

If you have issues with iMinuit try to install it from their github repo

## Testing

``pytest histogram``
